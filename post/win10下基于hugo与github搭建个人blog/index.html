My New Hugo Site

Win10下基于Hugo与Github搭建个人Blog

前置条件：已经安装Git

1. install Hugo

Hugo下载地址：https://github.com/gohugoio/hugo/releases

为了方便使用与管理，建议在无中文的路径下创建一个名为Github的文件夹，将下载好的Hugo文件夹统一放在里面。

Powershell » hugo version 查看版本号，若成功查看则表明安装成功。

2. 使用Hugo命令生成Blog

进入已经创建好的Github的文件夹，打开Powershell

Powershell » hugo new site myblog

3. 下载Hugo主题

Hugo主题库地址：https://themes.gohugo.io/

  • 方法一：进入myblog文件夹，打开Powershell，使用git克隆命令，具体命令在你选择点击的主题跳转的页面，例如：

  • 方法二：手动下载压缩包，解压后放进myblog下的themes文件夹，具体操作在你选择点击的主题跳转的页面点击Download后，将会跳转到相应的Github页面。

4. 本地启动Blog

在myblog下： Prowershell » hugo server -t [主题名称] --buildDrafts 在浏览器打开http://localhost:1313/

5. 写一篇测试博文

在myblog下：

Prowershell »hugo new post/xx.md 在myblog\content\post下创建一个.md文件，在此文件中编写你的博客内容
