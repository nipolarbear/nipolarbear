配置python篇

前置条件：

  • 安装好Python
  • 安装好vscode

安装vscode相关python插件

  • Python
  • Pylance
  • Code Runner （选装）
  • Jupyter （选装）

安装插件时可能会出现一些提示信息，有允许点允许，需重启vscode点重启vscode即可

配置settings.json文件

​ 打开设置

ctrl + ，（打开vscode设置快捷键）

​ 输入 setting，点击在settings.json 中编辑，进入settings.json配置文件

​ ctrl + z 撤回

因为点这个settings.json的时候是配置字体的，在此点进去会自动跳出字体的相关设置信息，不是我们需要配置的直接撤回去除，settings.json是vscode的配置文件无论什么插件或语言都是在此配置，并且是同一个settings.json。

​ 当打开settings.json的时候可能里面已经存在其中一些信息，我们不必删除，重点是在大花括号这项填入自己的pythonPath

配置launch.json文件

​ 打开一个文件夹，创建一个 01.py 测试文件，内容： print("hello, vscode")

​ 点击文件夹中的**.vscode**的文件夹，修改**launch.json**文件，

（如果没有此文件夹和文件可自行创建再复制粘贴即可）

​ 内容改为：

{
    // 使用 IntelliSense 了解相关属性。 
    // 悬停以查看现有属性的描述。
    // 欲了解更多信息，请访问: https://go.microsoft.com/fwlink/?linkid=830387
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Python",
            "type": "python",
            "request": "launch",
            "program": "${file}",
            "console": "integratedTerminal"
        }
    ]
}

​ 至此vscode的python环境配置已经完成，可运行01.py文件测试是否成功。

补充 关于Code Runner 插件无法输入问题

​ 打开设置，搜索 Code Runner，在到此项并且勾上，如果还是不行请多尝试几遍。
